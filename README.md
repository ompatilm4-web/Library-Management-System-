# Library Management 

A lightweight library management application built with Python and Tkinter.

## Features
- **Student Dashboard**: Browse available books and borrow them instantly.
- **Admin Dashboard**:
  - Add new students/members.
  - Add new books to the catalog.
  - View real-time library statistics.
- **Hardcoded Initial Data**: The app comes pre-loaded with sample students and books for immediate testing.

## Prerequisites
- **Python 3.x**
- **Tkinter**: On Ubuntu/Debian, install with:
  ```bash
  sudo apt-get install python3-tk
  ```

## How to Run
```bash
python3 main.py
```
